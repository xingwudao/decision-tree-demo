# 决策树学习演示

这是一个交互式的决策树学习演示项目，使用 Next.js 和 D3.js 构建。

## 功能特点

- 可视化决策树的构建过程
- 实时数据可视化
- 交互式预测工具
- 可调节的最小分裂样本数
- 动态的树结构调整

## 技术栈

- Next.js
- React
- D3.js
- Material-UI
- ML-CART (决策树算法)

## 本地开发

1. 克隆仓库：
   ```bash
   git clone https://github.com/xingwudao/decision-tree-demo.git
   ```

2. 安装依赖：
   ```bash
   cd decision-tree-demo
   npm install
   ```

3. 启动开发服务器：
   ```bash
   npm run dev
   ```

4. 打开浏览器访问 `http://localhost:3000`
